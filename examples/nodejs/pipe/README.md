# pipe

Whatever comes in must come out. Simple piping example: upload a file to POST /pipe.

```

```

```toml
addr = "127.0.0.1:9000"

[[http]]
addr = ":3000"

[[http.routes]]
path = "POST /pipe"
service = "pipe"
```

```js
const { Node } = require("flatend");

const main = async () => {
  await Node.start({
    addrs: ["127.0.0.1:9000"],
    services: {
      pipe: (ctx) => ctx.pipe(ctx),
    },
  });
};

main().catch((err) => console.error(err));
```
